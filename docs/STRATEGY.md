## 传统策略对比 (全部失败)

| 策略 | 结论 |
|------|------|
| VWAP 回归 | 信号太少, 2025→2026 不泛化 |
| 布林带 | 参数敏感, 跨年失效 |
| MACD | 滞后严重, 震荡市亏损 |
| RSI 超买超卖 | 阈值难调, 假信号多 |
| 多策略组合 | 互相干扰, 整体更差 |

**结论**: 技术指标策略在加密货币上表现差, 固定参数无法适应不同波动率环境。LightGBM 模型虽然 AUC 不高 (0.52), 但通过保守阈值过滤可以盈利。

## 策略引擎运行流程

```
每 60 秒:
  1. 获取最新 200 根 5m K线
  2. 计算 55 个特征
  3. LightGBM 预测概率
  4. 如果有持仓:
     a. 检查 TP/SL (用实时 ticker 价格, 不等下一根K线)
     b. 如果触发 → 平仓, 记录交易
  5. 如果无持仓:
     a. prob > 0.58 → 开多
     b. prob < 0.42 → 开空
     c. 计算仓位大小 (风险=资金×1%, ATR 定止损距离)
  6. 通过 OKX CLI 执行下单
  7. 记录到 trade_log + trades.json
```
