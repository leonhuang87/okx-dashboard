# OKX 交易系统开发文档

## 项目路径
`D:\软件制作\币交易开发\`

## 系统架构
```
前端 index.html ← 后端 server.py (Flask, port 5000) ← SQLite 数据库 (market.db)
                                                ← OKX 模拟盘 API (via CLI + Pilot 代理)
                                                ← LightGBM Model 01
```

## 启动方式
```powershell
# 1. 启动后端
python "D:\软件制作\币交易开发\server.py"

# 2. 浏览器打开
http://localhost:5000
```

## API 端点
| 端点 | 说明 |
|------|------|
| `GET /api/account/balance` | OKX 模拟盘余额 |
| `GET /api/account/positions` | 持仓列表 |
| `GET /api/account/fills` | 最近成交 |
| `GET /api/db/stats` | 数据库统计 |
| `GET /api/db/preview` | K线预览 |
| `GET /api/model/info` | Model 01 信息 |
| `POST /api/model/predict` | 实时预测 |
| `GET /api/backtest/results` | 回测结果 |
| `GET /api/status` | 系统状态 |

## Model 01 参数
- 标的: ETH-USDT 5m
- 训练期: 2025 Q1
- 标签: 1h 前瞻, 0.4% 阈值, 二分类
- 特征: 55 个技术指标
- 参数: conf>0.58, TP=3xATR, SL=2xATR, 最大持仓12根K线
- 手续费: 万分之3.6 双向
